Claw-Kanban

AI Agent Orchestration Kanban Board. Route tasks to Claude Code, Codex CLI, and Gemini CLI with intelligent role-based auto-assignment.

Features

⚖

Multi-Agent Routing

Automatically assign tasks to Claude Code, Codex CLI, or Gemini CLI based on role (devops/backend/frontend) and task type.

✓

Auto-Review

When an agent completes work, an automatic review is triggered. Pass review and the card moves to Done.

⚡

Gateway Integration

Optional OpenClaw gateway integration sends wake notifications on card status changes via WebSocket.

⌨

Terminal Viewer

Live streaming terminal output from running agents. Parses JSON streams from Claude, Codex, and Gemini.
